正解有两种

1.n^2预处理

由于n只有2000，我们可以n^2去预处理出答案，然后对于每次询问我们都O(1)回答
核心代码：
    for (i=0;i<n;i++){
        memset(f,0,sizeof(f) );
        x=0;
        for (j=i;j<n;j++){
            if ( f[a[j] ]==0 ) { f[a[j] ]=1; x++;}
            ans[i][j]=x;
        }
    }



2.前缀和思想
我们去维护sum[i][j]，表示[1,i]中，j究竟出现了多少次。
然后对于每一次询问，我们只要扫一遍m，判断sum[r][i]-sum[l-1][i]是否大于0就好了
这样预处理的复杂度是O(n)，对于每一个询问的复杂度是O(m)
所以总体复杂度是O(n+qm)
